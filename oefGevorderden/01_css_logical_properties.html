CSS Logical Properties

In hoofdstuk 9 kan je lezen over de nieuwe logische properties:

Huidig Nieuwe
*-top *-block-start
*-bottom *-block-end
*-left *-inline-start
*-right *-inline-end

waarbij het sterretje staat voor margin, padding of border.

Gebruik de Developer Tools om te weten te komen welke margin en padding, uitgedrukt in em en/of pixels, het ingebouwde stijlblad van je browser (user agent stylesheet) gebruikt voor de onderstaande elementen.

Maak hiervoor gebruik van het bestand speakers.html (uit les 3) en vul de door jou gevonden waarden in, in onderstaande tabel.

Opmerking: de font-size voor het body-element in de ingebouwde stylesheet is 16 pixels.
